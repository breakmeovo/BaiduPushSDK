百度云推送非官方，静态库下载至官方。

## BPush 版本
- version: 2.1.8

## 要求

- iOS 13.0+
- Xcode 9+
- Objective-C

## 安装
### CocoaPods

```ruby
target '<Your Target Name>' do
    pod 'RSDBPushSDK'
end
```
先执行 `pod repo update`，再执行 `pod install`。
